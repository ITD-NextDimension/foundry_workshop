agent=? endpoint=? token=?

👋 请用 launcher 启动

这个 chat UI 需要一个 AAD bearer token 才能调你的 hosted agent。学员手里只有 SP 凭据，launcher 会替你直接走 OAuth2 拿 token，不需要 az login。

正确启动方式(在 Lab-2-vibe-coding 目录, .env 里已填 SP):

cd Lab-2-vibe-coding
..\scripts\chat-hosted.ps1

脚本会自动:

  1. 从 .env / 进程 env / azd env 读 SP (AZURE_CLIENT_ID/SECRET/TENANT_ID) + endpoint + agent name
  2. 直接调 https://login.microsoftonline.com/<tenant>/oauth2/v2.0/token 做 client_credentials grant（不依赖 az / azd CLI）
  3. 把 endpoint / agent / token 注入 URL #cfg=，自动开浏览器

看到 launcher 报 "OAuth2 token 拿不到"? 多半是 .env 里 SP secret 没填对 / secret 已过期, 跟讲师确认。

实在不行才用右上角 ⚙ 高级 手填(适合调试 / 跨机器场景)。

手填只在调试场景需要。值来自学员 SP + AAD client_credentials grant (audience https://ai.azure.com/.default); 1 小时左右失效。不推荐手填 — 用 chat-hosted.ps1 它会自动注入。
勾上后 hosted agent 会在 Foundry 内置存储里记录这次会话(Lab 4 trace 用得上); 不勾 = stateless。
就绪。配置已从 URL 加载，或按 ⚙ 设置 手填。
输入消息按 Enter 发送，Shift+Enter 换行。
第一次用？先按右上角 ⚙ 设置 填好 endpoint / agent name / token。
